## 📌 Overview

CyberShield AI is an enterprise-grade AI-powered Cyber Resilience Platform designed to strengthen the security posture of Critical National Infrastructure (CNI).

Built for the **ET AI Hackathon 2026**, the platform enables Security Operations Centers (SOCs) to proactively detect cyber threats, investigate incidents, monitor behavioral anomalies, correlate threat intelligence, and generate executive security reports using Artificial Intelligence.

---

# 🎯 Problem Statement

**PS 7 – AI-Driven Cyber Resilience for Critical National Infrastructure**

Traditional SOCs struggle with:

- Slow threat detection
- Alert fatigue
- Manual investigations
- Lack of behavioral analytics
- Limited attack visibility
- High Mean Time To Detect (MTTD)
- High Mean Time To Respond (MTTR)

CyberShield AI addresses these challenges through intelligent automation and AI-assisted security operations.

# ⚡ Installation

Clone the repository

```bash
git clone https://github.com/your-username/cybershield-ai.git
```

Navigate to the project

```bash
cd cybershield-ai
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

---

# 🔧 Environment Variables

Create a `.env` file

```env
VITE_API_BASE_URL=http://localhost:5000/api
VITE_USE_MOCK_API=true
JWT_SECRET=your_secret_key
MONGODB_URI=your_mongodb_connection_string
```
